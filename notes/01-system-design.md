## What is System Design?

If you are asked to build a WhatsApp,

You may think:

- I will create a login page.
- Chat page.
- Store messages in a database.

That's **Software Development**.

But how will your system handle:

1. 3 billion users
2. 100 million users online at the same time
3. Millions of messages sent every second
4. The app should keep working even if one server fails

System Design changes the question from:

> **"How do I write code?"**

to

> **"How do I build an entire system?"**

<h3>Definition</h3>

> System Design is the process of deciding how different components of a software system work together to meet functional and non-functional requirements.

<h3>Goals of System Design</h3>

-  Fast
-  Secure
-  Reliable
-  Scalable
-  Easy to Maintain
-  Cost Effective
